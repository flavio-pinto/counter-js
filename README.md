Counter
===

Web app che permette l'incremento o il decremento di un counter con i pulsanti + e -, oltre a un reset che permette di tornare a zero

**Tecnologie usate**
===

* HTML
* CSS
* JavaScript

**Toolkits**
===

* Google Fonts
